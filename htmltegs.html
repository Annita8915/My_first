HTML теги

HTML — это язык гипертекстовой разметки страницы. Он состоит из тегов — команд браузеру, как отображать страницу
С помощью HTML делают текст жирным, курсивным, подчеркнутым, меняют его цвет. Ставят абзацы, обозначают заголовки, таблицы, списки. Размещают не только текст, но и ссылки или изображения.
HTML отвечает за структуру и контент — не за форматирование элементов страницы или динамику сайта. Поэтому разработчику нужно знать и другие технологии: CSS и JavaScript.

Вот основные HTML теги:


Базовые теги

<!--...-->

Комментарии

Помещённые внутри него теги не интерпретируются браузером

<!DOCTYPE>

Тип документа

Показывает браузеру тип документа, сообщает его версию и язык

<html> </html>

Корневой тег

Корневой тег, который сообщает браузеру, что это HTML-документ. Все остальные элементы помещаются внутри него

<head> </head>

Контейнер

Контейнер, в который помещаются метаданные документа, не видимые пользователям, но считываемые поисковыми роботами

<meta>

Оформление метаданных

Тег для оформления метаданных документа, используемых браузером для обработки страницы, а поисковиками — для индексации

<header> </header>

Цвет

Цвет фона документа в формате RGB

<body> </body>

Тело

Тег, обрамляющий видимую пользователям часть документа. Всё, что вы укажите внутри этого контейнера, отобразится на странице

<title> </title>

Название страницы

Метатег, который задаёт название страницы, отображаемое на вкладке браузера

<body bgcolor=?>

Содержимое с вводной информацией

Определяет содержимое блока с вводной информацией сайта или группой ссылок

Теги, форматирующие текст

<h1> </h1> ... <h6> </h6>

Заголовки

Теги заголовков, от самого большого к самому маленькому

<b> </b>

Жирное подчеркивание

Жирный текст без придания важности выделенному фрагменту

<strong> </strong>

Акцент

Расставление акцентов в тексте путём выделения его фрагментов полужирным начертанием

<i> </i>

Курсив

Выделение текста курсивом без придания важности

<del> </del>

Зачеркивание

Зачёркивает текст, помечая его удалённым

<ol> </ol>

Нумерованный список

Список с цифрами

<ul> </ul>

Нестрогий список

Список со значками

<li> </li>

Отметка элемента

Отметка каждого элемента перечня (цифра или значок в зависимости от типа списка)

<a> </a>

Гиперссылка

Добавление гиперссылки в текст. Имеет обязательный атрибут href, в котором указывается ссылка или якорь. Внутри контейнера помещается текст, при нажатии на который происходит переход на другую страницу или другое место на этой же странице

Теги, встраивающие элементы

<img> </img>

Изображение

Вставка изображения. Имеет атрибут src, который указывает на адрес нужного файла.

<img src="name" align=?>

Выравнивание

Выравнивание к одной из сторон документа. Например, значение right переместит рисунок в правый край, а left — в левый

<audio> </audio>

Аудио

Вставка звукового контента

<video> </video>

Видео

Вставка видео (поддерживается Ogg, WebM и MP4)

<source> </source>

Местоположение

Указывает местоположение файла для <video>, <audio> и <picture>

<track>

Субтитры

Формирует субтитры для <video> и <audio>

<object> </object>

Мультимедиа

Контейнер, через который встраиваются мультимедийные файлы. Для передачи параметров применяется тег <param> 

<hr> </hr>

Линия

Размещает на странице горизонтальную линию. Имеет несколько атрибутов 

<script> </script>

Сценарий

Определяет выполнение сценария на стороне посетителя сайта. Может содержать сам скрипт или иметь ссылку на внешний источник
